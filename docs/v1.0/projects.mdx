## URL

```
https://open.oneland.world/v1/projects
```

## Description
Get metaverse project list from OneLand platform.

## Type
GET

## Header

```
"Authorization": "Bearer <YOUR-API-TOKEN>"
```

## Sample Curl Command

```sh
curl -X GET 'https://open.oneland.world/v1/projects' \
  --header 'Authorization: Bearer <YOUR-API-TOKEN>'
```
