## URL

```
https://open.oneland.world/api/v1/project/{projectId}/floorPrices
```

## Description

Get a metaverse project's floor prices in paginated way. Floor prices will be indexed and calculated daily for each metaverse project.

Records will be returned **in date desending order**.

## Type

GET

## Header

```
"Authorization": "Bearer <YOUR-API-TOKEN>"
```

## Sample Curl Command
```sh
curl -X GET 'https://open.oneland.world/api/v1/project/{projectId}/floorPrices' \
  --header 'Authorization: Bearer <YOUR-API-TOKEN>'
```

## QUERY PARAMS

**offset**

For pagination. Number of record offset from the beginning of the result list. Default to 0.

**limit**

For pagination. Maximum number of records to return. Default to 10.
