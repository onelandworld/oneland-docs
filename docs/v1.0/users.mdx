## URL

```
https://open.oneland.world/v1/project/{projectId}/users
```

## Description

Get a metaverse project's users stats in paginated way. Users stats will be indexed hourly for each metaverse project.

Records will be returned **in time desending order**.

:::tip 
Currently only **Decentraland** is supported for this API
:::

## Type

GET

## Header

```
"Authorization": "Bearer <YOUR-API-TOKEN>"
```

## Sample Curl Command
```sh
curl -X GET 'https://open.oneland.world/v1/project/{projectId}/users' \
  --header 'Authorization: Bearer <YOUR-API-TOKEN>'
```

## QUERY PARAMS

**offset**

For pagination. Number of record offset from the beginning of the result list. Default to 0.

**limit**

For pagination. Maximum number of records to return. Default to 1. Maximum to 200.

## Sample Response

```json
{
    "code": 200,
    "message": "success",
    "data": [

            "timestamp": "2022-08-23T08:00:00.000Z"

        ...
```
