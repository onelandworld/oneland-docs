    URL#

    https://open.oneland.world/api/v1/project/{projectId}/stats

    Description#

    Get a metaverse project's stats data in paginated way. Stats will be indexed and calculated daily for each metaverse project.

    Records will be returned in date desending order.

    Type#

    GET

    Header#

    "Authorization": "Bearer <YOUR-API-TOKEN>"

    Sample Curl Command#

    curl -X GET 'https://open.oneland.world/api/v1/project/{projectId}/stats' \
    --header 'Authorization: Bearer <YOUR-API-TOKEN>'

    QUERY PARAMS#

    offset

    For pagination. Number of record offset from the beginning of the result list. Default to 0.

    limit

    For pagination. Maximum number of records to return. Default to 10.
